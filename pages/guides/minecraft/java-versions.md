В этой статье мы разберем выбор правильной версии Java для своего сервера. Бывало такое, что создал чистый новый сервер на 1.16.5, а он не хочет запускаться? Скорее всего, проблема в версии Java.

## Какую версию выбрать мне?

Наш хостинг предоставляет диапазон версий Java на выбор. Начиная с Java 8, заканчивая Java 25. Все эти версии требуются для разных версий самой игры Minecraft, и игра может не работать с определёнными версиями. 

Например, 1.16.5 поддерживает 8-ую, 11-ую Java, а Java выше может отказываться запускать сервер. В таблице ниже мы перечислили версии Java, требуемые для запуска той или иной версии Minecraft.

| Версия Java | Версия Minecraft |
| :--- | :--- |
| **Java 8** | Альфа-1.16.Х |
| **Java 11** | 1.0-1.16.5 |
| **Java 16** | 1.17 — 1.17.1 |
| **Java 17** | 1.18-1.20.4 |
| **Java 21** | 1.20.5 — 1.21.11 |
| **Java 22** | 1.21-1.21.11 |
| **Java 25** | Подойдёт для некоторых версий и ядер 1.21.Х, а также 26.Х |

## Что делать, если появляется ошибка несовместимости

Проверьте, что ваши плагины/моды не требуют Java выше, чем та, на которой работает сервер. Если так происходит, скорее всего, плагин не подходит для вашей версии, т.к. был сделан на более новой.

Ошибка совместимости также может возникать автоматически, если у вас сервер сам поймет то, что версия Java не сходится с версией Minecraft. Чаще всего, автоматически появляется на Paper.

## Почему лучше использовать современные версии Java?

Современные версии Java намного лучше работают с оперативной памятью и быстродействием самого сервера Minecraft.

Для разработчиков плагинов - современная Java легче, и работать с ней намного приятнее. Синтаксис стал проще, а функций больше
