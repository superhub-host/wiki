В этом гайде мы разберём, как создать сервер на хостинге и правильно подобрать тариф под твои нужды.

## С чего начать?

Для начала определись, каким будет твой сервер: Minecraft, Hytale или VPS? Самый важный этап — выбор тарифа и ядра.

Для серверов Minecraft у нас поддерживаются разные предустановки ядер, для Hytale — только официальное, для VPS — разные образы операционных систем.

Перейди на страницу создания сервера. Все наши услуги описаны на [главной странице](/), либо можно использовать прямые ссылки:

| Услуга |
| :----- |
| [Хостинг Minecraft](/order/minecraft) |
| [Хостинг VPS](/virtual-hosting) |
| [Хостинг Hytale](/hytale-hosting) |

Следуй инструкциям, предложенным в форме.

## Тариф

### Тарифная линейка

При покупке сервера тебе будет предложено выбрать тарифную линейку. От неё будет напрямую зависеть качество предоставляемых услуг: доступные ресурсы, производительность и приоритет на ноде. О том, как выбрать подходящий вариант, можешь прочитать в [отдельной статье](/guides/hosting/tariff-sets).

### Период оплаты

После выбора тарифной линейки нужно выбрать подходящий тебе платёжный цикл.

![Выбор периода оплаты](/images/guides/hosting/order-server/billing-period-picker.png)

Список возможных вариантов может меняться в зависимости от услуги. Например, VPS можно оплачивать только ежемесячно, а на хостинге Hytale пока недоступны вечные сервера. Возможные варианты:

- **Единоразово** — ты платишь за сервер только при покупке и можешь использовать его неограниченное количество времени. Доплатить придётся только если захочешь перейти на более мощный тариф.
- **Ежемесячно** — ты платишь при покупке, а затем каждые 30 календарных дней. Стоимость может измениться, только если ты используешь промо-код с ограниченным сроком действия или решишь сменить тариф.
- **Ежедневно** — ты платишь при покупке, а затем каждые 24 часа. Стоимость может измениться, только если ты используешь промо-код с ограниченным сроком действия или решишь сменить тариф.

!> Совет: если тебе нужен сервер надолго, выгоднее выбрать **ежемесячную** оплату — месячный тариф обойдётся дешевле, чем 30 дней подневного. Если всё же берёшь сервер с подневной оплатой, лучше пополняй баланс сразу на крупную сумму: при небольших пополнениях может взиматься дополнительная комиссия.

### Выбор тарифа

В зависимости от выбранного периода оплаты тебе будет доступен либо ручной выбор ресурсов (конфигуратор), либо готовые тарифы:

| Услуга            | Период оплаты | Формат выбора конфигурации |
| :---------------- | :------------ | :------------------------- |
| Minecraft         | Единоразово   | Конфигуратор               |
| Minecraft, Hytale | Ежемесячно    | Готовые тарифы             |
| Minecraft, Hytale | Ежедневно     | Конфигуратор               |
| VPS               | Ежемесячно    | Конфигуратор               |

Если выбираешь сервер для Minecraft, придерживайся [наших рекомендаций](/blog/server-resources) по выбору тарифа.

## Локация

После выбора тарифа тебе будет предложено выбрать ноду, где будет располагаться сервер. Как правило, стоит выбирать ту, которая географически находится ближе всего к большинству твоих игроков — это напрямую влияет на пинг и качество игры.

?> Учитывай, что если ты находишься в России, ноды в Европе могут быть недоступны из-за ограничений со стороны регулирующих органов. Если находишься в Украине, наоборот — лучше выбрать европейскую локацию из-за возможных проблем с доступом к российским интернет-ресурсам.

Отнесись к выбору локации внимательно. Чтобы изменить её впоследствии, нужно будет написать в техническую поддержку. Подробнее об изменении локации и тарифа — в [отдельной статье](/guides/hosting/change-location-or-tariff).

## Предустановленное ПО

### Игровые сервера

При покупке тебе будет предложено выбрать ядро и версию игры, которые будут автоматически предустановлены на сервер. Впоследствии и ядро, и версию [можно изменить](/guides/change-version) или вовсе установить другое ядро, даже если оно недоступно в форме покупки.

Если хочешь играть с модами — выбирай Forge или Fabric. Если хочешь создать сервер с плагинами — Paper или Purpur.

Все варианты и их назначение описаны в таблице:

| Характеристика | **Ядра** (Java) | **Ядра** (Bedrock) | **Ядра** (Прокси) |
| :--- | :--- | :--- | :--- |
| **Нужны моды** | Forge, Fabric, NeoForge, Quilt | — | Velocity, Bungeecord, Geyser |
| **Нужны плагины** | Paper, Purpur, Folia | PocketMineMP, PowerNukkitX, Endstone | Velocity, Bungeecord, Geyser |
| **Нужны аддоны** | — | Vanilla | Geyser (для Bedrock) |

### VPS

При покупке тебе будет предложено выбрать ОС, которая будет предустановлена на сервер. Впоследствии её можно переустановить через панель управления.

Выбери наиболее удобный для тебя дистрибутив Linux и подходящую версию. Если сомневаешься — бери Ubuntu последней версии, это отличный вариант для начинающих.

## Завершение покупки

Почти готово — осталось несколько финальных шагов.

### Промо-код

Если у тебя есть промо-код, укажи его в форме — для этого потребуется авторизоваться на сайте. Если промо-кода нет, просто переходи дальше.

Затем придумай название для сервера. Оно будет отображаться только в панели управления и личном кабинете, игроки его не увидят. Название нужно только для того, чтобы ты сам не запутался, если у тебя будет несколько серверов.

!> **Убедись**, что на балансе уже есть средства для покупки. Если их не хватит, система попросит пополнить баланс, и после этого все шаги придётся пройти заново.

В конце нажми «Создать сервер». Как только всё будет готово, тебя перенаправит в личный кабинет, где ты сможешь управлять сервером.
